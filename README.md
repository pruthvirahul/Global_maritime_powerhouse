# ⚓ Maritime Powerhouse

**Real-time Advanced Naval Assets Analytics**

Maritime Powerhouse is a comprehensive, real-time web application designed to analyze, visualize, and compare global naval capabilities. It provides deep strategic insights into the world's most powerful fleets and their missile arsenals.

## 🌟 Key Features

*   **📊 Global Power Rankings**: Computes and ranks nations based on their overall naval combat power, factoring in fleet size, technological generations, and missile synergy.
*   **🚢 Fleet Analytics**: A detailed, filterable intelligence grid of the world's most advanced warships and submarines (e.g., Arleigh Burke, Yasen-M, Type 055), displaying their firepower, defense, and integrated weapons systems.
*   **🚀 Missile Arsenal**: A massive database of 20+ strategic and tactical missile systems spanning the USA, Russia, China, India, UK, and France. Includes advanced threat-level sorting.
*   **🔬 Strategic Upgrade Lab**: An advanced tactical view that analyzes current missile capabilities against a 100% theoretical maximum.
    *   **Multi-Currency R&D Estimation**: Automatically calculates upgrade costs in USD, INR (₹), RUB (₽), and CNY (¥) with exchange rates.
    *   **AI Recommendations**: Generates strategic advice such as "Evasion-Speed Mismatch" or "Full Platform Overhaul Required" based on dynamic stat analysis.

## 🛠️ Technology Stack

*   **Frontend**: Vanilla HTML, CSS, JavaScript (ES6+ Modules)
*   **Build Tool**: Vite (for lightning-fast HMR and optimized production builds)
*   **Backend**: Node.js & Express (Lightweight API server to serve JSON intelligence data)
*   **Styling**: Custom CSS with Glassmorphism, CSS Variables, and advanced keyframe animations (No external UI libraries used).

## 🚀 Getting Started (Local Development)

### Prerequisites
*   [Node.js](https://nodejs.org/) (v16 or higher recommended)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/pruthvirahul/Global_maritime_powerhouse.git
    cd Global_maritime_powerhouse
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server and backend API simultaneously:
    ```bash
    npm run start
    ```

4.  Open your browser and navigate to:
    ```text
    http://localhost:5173
    ```
